HTML是什么

Hyper Text MarKup Language
HTML 不是一个编程语言丶是标记语言

标记语言的作用是制作网页结构或者文档

HTML是前端最重要的同时也是最基础的部分

HTML阶段会相对比较枯燥并且内容很丑

css是什么

Cascoading Style Sheets
css是一个样式语言，层叠样式表

css是针对于HTML标签设计样式

css也可以扩展为Sass/Less

Sass/Less是css的脚本语言
会有专门的课程

网页各个技术的分工

HTML = 内容展示/结构

css = 样式/设计

JavaScript = 前端编程语言 "让网页动起来"

PHP，Python，c# = 后端编程语言 "处理数据"

MySQL，MongoDB = 数据库 "存储数据"

只有HTML&css能做什么

制作静态网页
个人简历，商品展示等
